# Sapient — Swift Package

**On-device LLM, vision, and speech inference for iOS & macOS**, powered by
the pure-Rust [SAPIENT](https://github.com/SkidGod4444/sapient) engine —
GPU by default (wgpu → Metal, CPU fallback), engine-level thermal governance,
streaming with cancel.

## Install

Xcode → *File → Add Package Dependencies* → paste this repo's URL:

```
https://github.com/openhorizon-labs/sapient-swift
```

`Package.swift` points a checksum-pinned `binaryTarget` at the matching
SAPIENT release's `SapientFFI.xcframework.zip`, so SwiftPM downloads and
verifies the binary itself. Each SAPIENT release re-points this package and
tags it with the same version.

## Use

```swift
import Sapient

// Call off the main thread — load() blocks and downloads on first run.
let session = try LlmSession.load(model: "qwen2.5-0.5b",
                                  options: GenerationOptions(maxTokens: 256))
let reply = try session.chat(userMessage: "Hi!")
```

Full guide (streaming, thermal wiring, the safe-testing ladder for personal
devices): [docs/MOBILE.md](https://github.com/SkidGod4444/sapient/blob/main/docs/MOBILE.md).
